# Traffic-Light_splunk_app
Traffic Light app for Splunk 5

This app is re-write of Traffic Light Visualisations app by David Greenwood to work with Splunk 5 version.Thanks to David Greenwood:)

Installation: 

1. Untar the Traffic-Light_splunk_app.tar.gz file in your $SPLUNK_HOME/etc/apps directory.
2. Restart Splunk.
